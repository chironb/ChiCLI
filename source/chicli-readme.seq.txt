ChiCLI Read Me

ChiCLI (c) 2022 Chiron Bramberger
A CLI for Commodore Computers!

A Commodore 64 command line
interface terminal, similar to and
inspired by: AmigaDOS, Commodore DOS,
MS-DOS, & Linux.

ChiCLI Pronunciation: ky see-elle-eye

For more information and source code:
GitHub.com/ChironB/ChiCLI

To view this file in ChiCLI, enter:
type chicli-readme

To get help, enter:
help

To get a list of commands,
hit F8 or enter:
help commands

To read the manual, enter:
type chicli-manual

Thanks for using ChiCLI!
- Chiron Bramberger
